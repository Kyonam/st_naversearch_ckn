# 네이버 오픈API 애플리케이션 등록 가이드

네이버 오픈API를 사용하기 위해 가장 먼저 수행해야 하는 애플리케이션 등록 과정을 상세히 설명합니다.

## 1. 애플리케이션 등록 단계
1. **네이버 개발자 센터 접속**: [developers.naver.com](https://developers.naver.com)에 로그인합니다.
2. **이용신청 페이지 이동**: 상단 메뉴에서 `Application > 애플리케이션 등록` 메뉴를 선택합니다.
3. **약관 동의**: API 이용 약관에 동의합니다.
4. **계정 정보 등록**:
    - 휴대폰 인증 (담당자 확인을 위해 최초 1회 필수)
    - 회사 이름 (개인 개발자는 이름 입력 가능)
5. **애플리케이션 상세 정보 입력**: 아래 섹션의 내용을 입력합니다.
6. **등록 완료**: `등록하기` 버튼을 클릭하면 Client ID와 Client Secret이 즉시 발급됩니다.

## 2. 애플리케이션 등록 상세 정보
### 2.1. 애플리케이션 이름
- 네이버 로그인 시 사용자에게 노출되는 이름입니다 (최대 40자).
- 권장: 10자 이내의 간결한 이름.

### 2.2. 사용 API 선택
- 애플리케이션에서 사용할 API를 목록에서 선택합니다 (예: 검색, 데이터랩, 네이버 로그인 등).
- **중요**: 여기서 선택하지 않은 API는 코드에서 호출 시 `403 Forbidden` 오류가 발생합니다.

### 2.3. 서비스 환경 설정
API 호출을 허용할 환경을 추가해야 합니다.
- **WEB 설정**: 서비스할 웹 도메인을 입력합니다 (예: `http://localhost:3000`, `https://example.com`).
    - 도메인 입력 시 서브도메인을 포함하지 않아도 하위 도메인이 자동 포함될 수 있습니다.
- **Android 설정**: 안드로이드 패키지 이름을 입력합니다.
- **iOS 설정**: iOS Bundle ID를 입력합니다.

## 3. Client ID 및 Client Secret 확인
- 등록 완료 후 `Application > 내 애플리케이션` 메뉴에서 해당 앱을 클릭합니다.
- **개요** 탭에서 `Client ID`와 `Client Secret`을 확인할 수 있습니다.
- `Client Secret`은 비밀번호와 같으므로 외부에 노출되지 않도록 주의하십시오. 필요 시 `재발급` 버튼을 통해 갱신할 수 있습니다.

## 4. API 설정 변경
- 등록 이후에도 언제든지 사용 API를 추가/삭제하거나 서비스 도메인을 업데이트할 수 있습니다.
- `API 설정` 탭에서 설정을 변경한 후 `수정하기`를 클릭하면 즉시 반영됩니다.
